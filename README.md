# 氢刻知间 Newsletter

我开启这个 Newsletter 的动机：

1. 练习写作
2. 督促自己持续输入高质量信息
3. 希望通过分享对读者可能有价值的内容获得一点正反馈，然后让自己正向循环起来

为了能更好地坚持下去，我给了自己一个很低的门槛：选择了分享链接 + 简评的方式，将自己认为好的内容分享出去，顺便留下一两句简评。

本 Newsletter 开源、首发在 [GitHub](https://github.com/rokcso/weekly)。
